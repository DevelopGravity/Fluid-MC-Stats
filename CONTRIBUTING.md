# Contributing

## Rules

- Do not spam comment.
- Do not bump issues.
- Make sure you are using the latest version of both, the plugin and interface.
- Attach images of the errors.
- Copy the error text if any.
- Open `config.php` and set debug to `true`. Copy the output on the site.
